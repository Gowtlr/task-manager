# Task Manager

Projeto de gerenciador de tarefas feito com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar como concluídas
- Definir prioridade
- Definir prazo
- Visualizar tarefas atrasadas
- Filtrar tarefas
- Salvar dados com localStorage

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
