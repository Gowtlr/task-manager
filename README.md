# Task Manager

Projeto de gerenciador de tarefas feito com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar como concluídas
- Definir prioridade
- Definir data e horário
- Buscar tarefas com autocomplete
- Alternar entre modo claro e escuro
- Visualizar tarefas atrasadas
- Filtrar tarefas
- Salvar dados com localStorage
